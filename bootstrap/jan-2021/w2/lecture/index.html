Bootstrap Setup

  • create a directory for your project
  • navigate to directory using terminal
  • $ npm init -y
  • $ code . (to open directory to VS Code)
  • $ npm i lite-server@2.5.4 --save-dev
  • $ npm i jquery@3.5.1 popper.js@1.16.1
  • $ npm i bootstrap@4.5.2
  • Edit scripts in package.json with: "scripts": { "lite": "lite-server", "start": "npm run lite", "test": "echo \"Error: no test specified\" && exit 1" },
  • Make sure you are in the correct directory in the terminal. Create index.html file using $ code index.html
  • Inside the index.html file type "!" then enter to setup basic html file
  • viewport meta tag
  • link Bootstrap CSS file
  • link scripts for jquery, popper, bootstrap
  • run $ npm start
