• $ git branch - show you all your branches and it will display which one you are currently on, you can also see which branch you are on with vs code on the bottom left corner it will tell you the branch name
  • $ git branch branch-name - create a branch
  • $ git branch -D branch-name - delete a branch
  • $ git checkout branch-name - move between branches ***before moving to another branch make sure you git add, commit, and push origin branch-name***
