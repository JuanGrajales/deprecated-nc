Setup directory and package.json

  1. Create nucampsite directory in bootstrap directory
  2. Inside nucampsite directory run $ npm init -y

    3. Install dependencies

  3. Inside nucampsite directory run
    $ npm i lite-server@2.5.4 --save-dev
    $ npm i jquery@3.5.1 popper.js@1.16.1
    $ npm i bootstrap@4.5.2

    4. Modify package.json

  4. Edit package.json file with the following:
    "description": "This is a website for the fictional campsites review site NuCamp.", "main": "index.html", "scripts": { "lite": "lite-server", "start": "npm run lite", "test": "echo \"Error: no test specified\" && exit 1" },

    5. Create homepage

  5. Create an index.html file and add the code provided
  6. In your terminal find the location of the package.json file and run $ npm start

    7. Add bootstrap

  7. Replace the HTML code in the head element
  8. Add scripts provided under the footer element

    9. Option: Git

  9. Create a .gitignore file and add "node_modules"
